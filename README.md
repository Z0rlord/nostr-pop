# nostr-pop

NOSTR-native proof-of-practice — sword practice videos from YouTube to Blossom
and Nostr (YakiHonne-compatible NIP-71 kind 22).

## Quick start

```bash
doppler run -- uv run --project pipeline pipeline/pipeline.py \
  --url <youtube-url> --max-duration 90
```

See [AGENTS.md](./AGENTS.md), [pipeline/README.md](./pipeline/README.md), and
[docs/sessions/](./docs/sessions/) for layout, relays, and session history.
