# 真の位

**中段** — 胴の高さの拳。

| 位 | 備考 |
|----|------|
| **青眼の位** | 柄頭水月、切っ先両眼間；静眼／勢眼の使い分け |
| **片目外しの青眼** | 誘い返 |
| **楯の位（真）** | 中峯；胸鍔高；堅守しつつ攻めやすい |
| **蛇躰の位（真）** | 天の蛇躰より前傾控えめ |
| **軒端覗の位** | 軒端から覗く |
| **肩掛の位** | 残心；袈裟外し後など |
| **亀袖柄** | 亀の如く袖に拳；間合制御 |
| **楓隠の位** | 左手楓葉で誘う |
| **横笛太刀（真）** | 待ち位；上中下段防御 |
| **弓勢獅子舞** | 泥濘・古伝介者；爺婆腰 |

**青眼** は十二勢法の定番残心位。
