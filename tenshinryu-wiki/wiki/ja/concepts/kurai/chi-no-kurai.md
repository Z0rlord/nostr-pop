# 地の位

**下段** — 胸より下。待ち・誘い・低線防御。

| 位 | 役割 |
|----|------|
| **水月の位** | 待ち；車輪・峰谷；柄頭のみ見せる |
| **地摺の青眼** | 切っ先地を摺る；誘い；持久 |
| **地の位崩し** | 地摺から左右に切っ先外し |
| **柄落胸刀** | 柄落とし胸刀；靶面最小化 |
| **軒端掛の位** | 軒端に掛ける如く；深い誘い |
| **横笛太刀（地）** | 下段待ち |
| **楯の位（地）** | 逆楯；杠杆抜刀 |

半身・根ノ足・前傾が多い。
