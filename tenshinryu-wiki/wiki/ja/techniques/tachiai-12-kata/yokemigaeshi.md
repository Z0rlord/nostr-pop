# 五 — 避身返・逃身返

![勢法五：避身返・逃身返（よけみがえし・にげみがえし）](/assets/tachiai-12-kata/yokemigaeshi/%E9%99%B0/DSC_0244.JPG)

| | |
|---|---|
| **隠し名** | 畷太刀（なわてだち） |
| **分類** | 畦道・両手荷物 |

## 想定

田んぼの **畦道（畷）** を歩き、前方の敵が抜刀して走りかかる。両手に荷物。

## 流れ

1. 荷を落とし、右回りに踵を返し **柳生腰**・**虎走** で逃走
2. 十字路手前で減速し攻撃を誘う
3. 切り来る刹那、陰陽転じ抜刀反撃

**陰転：** 鞘事天地返し、袈裟外し → 肩返しで二ノ太刀  
**陽転：** 左踏切・左回り反転、袈裟懸け、右足前着地

大袈裟 → 八草 → 青眼 → 納刀。

避ける身／逃げる身から **避身返**・**逃身返**。
