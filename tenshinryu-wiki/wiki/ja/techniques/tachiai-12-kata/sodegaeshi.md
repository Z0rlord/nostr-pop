# 六 — 袖返（そでがえし）
![勢法六：袖返（そでがえし）](/assets/tachiai-12-kata/sodegaeshi/%E6%8A%9C%E8%A8%8E/DSC_0267.JPG)

| | |
|---|---|
| **隠し名** | 野分太刀（のわけだち） |
| **分類** | 護身・抜討 |

## 護身

後方から左手で鞘尻、右手で右袖を掴まれ抜刀不能。刀を大きく動かし力を弱め、陰陽転じて鞘事天地返し抜刀・反撃。追撃か退きかは自由。**転柄血振い**・**逆手納刀** 定めあり。

## 抜討（暗殺）

細道の標的を後方追跡 → 追い抜き → 気緩みに左回り反転、顎を真っ向切り上げ、面を切り下す。雲居 → 切雲 → 青眼。血振・納刀同上。

**袖返：** 袖がひるがえる陰陽転じに由来。  
**野分太刀：** 野を分ける風の如く身を転じる。
