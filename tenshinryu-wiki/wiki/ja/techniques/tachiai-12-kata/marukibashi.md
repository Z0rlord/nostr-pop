# 九 — 丸木橋（まるきばし）

![勢法九：丸木橋（まるきばし）](/assets/tachiai-12-kata/marukibashi/%E4%B8%B8%E6%9C%A8%E6%A9%8B.jpg)

| | |
|---|---|
| **隠し名** | 陰陽一入（いんよういっそう）、狭間返（はざまがえし） |
| **分類** | 吊り橋・不安定足場 |

## 想定

丸太を紐で吊るす **丸木橋**（同様に廊下など左右に振れない場所も含む）。前後挟み撃ち、敵は天の位。

**柳生腰** で左右を配る。右敵の真向切り下ろし → 右向き直し **摺刃太刀**、切っ先返し小手切り。左敵 → 左振返り、刃下・峰上で受け、小手切り下ろし。

左肩掛け → 右回りで最初の方向 → 大息 → **青眼**。息止め。**転柄血振い**・**逆手納刀**。

足運びは初学・次段・実用で撞木足の使い方が異なる。
