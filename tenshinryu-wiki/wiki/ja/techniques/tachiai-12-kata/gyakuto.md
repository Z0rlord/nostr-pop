# 八 — 逆刀（ぎゃくとう）

![勢法八：逆刀（ぎゃくとう）](/assets/tachiai-12-kata/gyakuto/%E9%80%86%E5%88%80.jpg)

| | |
|---|---|
| **隠し名** | 柄下虎落太刀、逆刃柄返し |
| **分類** | 極近間合・柄取り |

## 柄を取らせない

左手鞘、右手 **逆手** 柄。右肩上に抱え右斜め後ろ **撞木足** → 高抜 → 頭上で順手持替 → 左足引き袈裟懸け → 六尺以上 **雲居** → 切雲 → 青眼。**転柄血振い**・**逆手納刀**。

## 掴まれた場合

柄を上下左右捻って離す → 以降同じ。

**逆刀：** 逆手抜刀。**柄下虎落太刀：** 虎落笛の如く太刀鳴り。**逆刃柄返し：** 妙手返しで柄持替。
