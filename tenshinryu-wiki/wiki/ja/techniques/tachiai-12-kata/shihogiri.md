# 七 — 四方切（しほうぎり）

![勢法七：四方切（しほうぎり）](/assets/tachiai-12-kata/shihogiri/%E5%88%A5%E3%82%A2%E3%83%B3%E3%82%B0%E3%83%AB.jpg)

| | |
|---|---|
| **隠し名** | 柵太刀（しがらみだち）、袈裟乱れ（けさみだれ） |
| **分類** | 四方囲み |

## 配置

後方の敵のみ未抜刀。他三人は天の位。左足前、柄頭高く **柳生腰** で前進。

## 四段（順）

1. **後（1）：** 抜刀企て → 右捻り抜刀、右片手袈裟で右前腕
2. **正（2）：** **摺刃太刀** で受け外し → 左肩口袈裟、切っ先地面すれすれ
3. **右（3）：** 振りかぶる前に右足引き、左胸から右脇下へ
4. **左（4）：** 左軸で低く飛び、左肩・左腕引き切り、右足前着地

左 **肩掛けの位** → 右足軸で右回り正面 → 大息 → **青眼**。開始から息止め。**転柄血振い**・**逆手納刀**。

後方の敵に隙を見せて切り掛からせる心持ちが重要。
