# 十 — 二ノ太刀返

![勢法十：二ノ太刀返（にのたちがえし）](/assets/tachiai-12-kata/ninotachigaeshi/%E4%BA%8C%E3%83%8E%E5%A4%AA%E5%88%80%E8%BF%94.jpg)

| | |
|---|---|
| **隠し名** | 縺太刀、立合斬留之事、切留後先返 |
| **分類** | 脛切りからの連続攻撃 |

## 想定

片手で **右脛** を切りつけられる → 対者の二ノ太刀（面切り）へ。

## 要点

1. 左足横移動、右膝高く **鶴の足**、切っ先床刺しの如く **十字受**
2. 対者は途中で止め面切り → 切っ先左から上へ **かぶせ受け**
3. 右膝横開き、左膝地突き、**一角絡柄**・**蟷螂返** で前腕切り上げ
4. 左足前出し真向切り下ろし追撃
5. 雲居 → 切雲 → 青眼 → 納刀

刀を縺れさせる如く受ける → **縺太刀**。
