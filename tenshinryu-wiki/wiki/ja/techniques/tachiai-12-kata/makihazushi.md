# 四 — 巻外・巻落

![勢法四：巻外・巻落（まきはずし・まきおとし）](/assets/tachiai-12-kata/makihazushi/%E5%90%8D%E7%A7%B0%E6%9C%AA%E8%A8%AD%E5%AE%9A%201.jpg)

| | |
|---|---|
| **隠し名** | 月影（つきかげ） |
| **分類** | 防刀 |

## 想定

片手袈裟懸けで頭・肩口を狙われる。

## 要点

1. 左右いずれかの足を引き抜刀、鎬・峰で受ける
2. 身を左に捻り **時計回り** に刀を巻き外す（巻外・巻落）
3. 敵の右小手を右から左へ袈裟懸け
4. 大袈裟の位 → 八草の位 → 青眼 → 納刀

**月影：** 自刀を月、敵刀を雲と見立て巻く。
