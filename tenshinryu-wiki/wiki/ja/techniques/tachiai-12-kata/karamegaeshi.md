# 十一 — 絡返（からめがえし）

![勢法十一：絡返（からめがえし）](/assets/tachiai-12-kata/karamegaeshi/%E5%88%A5%E3%82%A2%E3%83%B3%E3%82%B0%E3%83%AB.jpg)

| | |
|---|---|
| **隠し名** | 不動絡太刀（ふどうからめだち）、腰車（こしぐるま） |
| **分類** | 水平切り |

## 想定

片手で水平に肩口・首・目・顳などを狙われる。

## 要点

1. 切っ先上 **不動剣**、左足斜め後ろ、鎬で受け、両足 **撞木足**
2. 刀を右に旋回して絡める
3. 柄頭を胸中央へ、左手を柄頭に
4. 右足前、柄頭を押して右腕・右脇下を切り刺す（大前傾）
5. 左足追撃、**捲り切り** で面押し切り
6. 雲居 → 切雲 → 青眼 → 納刀

**腰車：** 腰で太刀を車の如く返す。
