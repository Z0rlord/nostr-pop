# 一 — 面陰（おもかげ）

![勢法一：面陰（おもかげ）](/assets/tachiai-12-kata/omokage/%E9%9D%A2%E9%99%B0.jpg)

| | |
|---|---|
| **隠し名** | 傘雪剣（さんせつけん） |
| **分類** | 防刀 |

## 想定

立ち・歩行中、突然真っ向（或いは袈裟懸け）切り掛かる。

## 要点

1. やや右斜前に出つつ抜刀
2. 柄を右前腕につけ、肘近くの前腕を右額に当て **鎬・峰** で受ける（傘に雪＝傘雪剣）
3. 面を陰にして目を刃から守る → **面陰**
4. 返刃で肩口・三角筋と上腕二頭筋の間を袈裟引き切り
5. **大袈裟の位** 或いは **青眼の位** で六尺以上退き、残心・納刀

## 術歌

> 雪の道 傘に積りし重き柄（え）も 捻じり閉じれば舞い散る傘雪（かさゆき）
