# 天心流兵法

**天心流**（てんしんりゅう）は、江戸時代初期（約400年前）に起源を持つ日本伝統の**兵法**（総合武術）である。剣・槍・鎖武器・裸手を統合し、寝起き・座立・歩行など日常の姿勢から応用できる戦術を重視する。

関連: [[history/overview]] · [[arts/_index]]

## 概要

| | |
|---|---|
| **開祖** | 時沢弥兵衛（柳生宗矩の下で体系化） |
| **時代** | 寛永期；新陰流等を基に編成 |
| **歴史上の目的** | 徳川幕府を影で支えた**光願**（こうがん）の武士のための術 |
| **現系譜** | 8代石井清造 → 9代中村天心 |
| **現代** | 関東の道場網；海外支部；KIWAMI によるオンライン稽古 |

## 「兵法」としての位置づけ

単一の武器に特化した流派ではなく、八つの主要な**art**を含む（[[arts/_index]] 参照）。**殿中刀法鞘之中**、**古伝**の太刀形、常時**脇差**を帯する稽古など、武士の生活前提が組み込まれている。

## 今日の学び方

- **国内**: 関東を中心に複数の稽古場 — 公式サイト参照。
- **海外**: 連盟・支部；[Tenshinryu KIWAMI](https://tenshinryu.xyz) によるオンラインカリキュラム。
- **抜刀核心**: [[techniques/tachiai-12-kata]] — 立相十二勢法。
