# Sanmi-no-Kurai (三躰の位)

Collective name for Tenshinryu's **three basic ways of standing** — also **ChikuSyaRoboku (竹斜老木)**. With sword in hand: **Sanmitachi-no-Kurai (三躰太刀の位)**.
No single stance dominates real combat; **balance practice** across all three.
| Standing body | Page | Role (summary) |
|---------------|------|----------------|
| **Robokudachi** 老木立 | [[concepts/kurai/robokudachi|Robokudachi 老木立]] | Armor-era standard; wide, deep knees; chi-tier foot width reference |
| **Shabokudachi** 斜木立 | [[concepts/kurai/shabokudachi|Shabokudachi 斜木立]] | Edo / suhada line; angled tree; shin-tier default width |
| **Chikubokudachi** 竹木立 | [[concepts/kurai/chikubokudachi|Chikubokudachi 竹木立]] | Owari Yagyu upright bamboo; beginners' upper body; far maai from drop |
