# 3 — Nukidome (抜留)
![Seiho 3: Nukidome 抜留](/assets/tachiai-12-kata/nukidome/%E6%8A%9C%E7%95%99.jpg)

| | |
|---|---|
| **Hidden name** | Bakuchiken 驀地剣 |
| **Type** | Stop opponent's draw |

## Situation

Opponent reaches for tsuka to draw; cut as if **thrusting at forearm** (小手).

## Variations

- Normal standing
- From **Yagyugoshi**
- Run in from distance in Yagyugoshi

## Movement

1. Begin almost straight; later **diagonal forward-right** (in-yo)
2. Unlike Nukiai — **no upward cut**; thrusting cut hidden behind tsuba
3. Return to **Katamehazushinoseigan**-like posture; **tsume** — tip at throat
4. On surrender → retreat ≥180 cm **Seigannokurai** → zanshin → noto

**Bakuchiken:** furious forward lean in **Nigurumadachi** — rushing straight.

> Distinguish carefully from [[techniques/tachiai-12-kata/nukiai|Nukiai]] (overlapping scenario).
