# 2 — Nukiai (抜合 / 抜相)
![Seiho 2: Nukiai 抜合](/assets/tachiai-12-kata/nukiai/%E5%90%8D%E7%A7%B0%E6%9C%AA%E8%A8%AD%E5%AE%9A%201.jpg)

| | |
|---|---|
| **Hidden names** | Kusamadachi, Kuchinotachi, Taijiken, Ichiedachi, Nigurumadachi, Kurumadachi |
| **Type** | Mutual draw — standing duel |

## Situation

Face enemy; both draw; cut **straight upward** (真っ向).

## Stages

1. **Beginner:** Pull left or right leg back; draw; upward cut (slight kesahazushi OK — lower-left to upper-right)
2. **Next:** Approach walking; draw straight or **in-yo** turn immediately before
3. **Advanced:** Run in; optional **Hisen** (fan throw) before draw

## Second cut

Return tip from left, or **Chitengaeshi** — cut straight down to head.

## Finish

Large retreat → **Kumoinokurai** → **Sekiunnokurai** → zanshin → **Seigannokurai** → zanshin → noto.

## Notes

- Beginner posture = **Nigurumadachi** (荷車太刀 — cart-pusher stance)
- Historically taught **first** before Omokage (JP text)
